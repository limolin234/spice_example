```bash
micromamba run -n sci python ./sim/sim.py
```

## Workflow

- `cir/main.cir`: human-written circuit topology.
- `cir/main.par`: circuit parameters generated by Python.
- `sim/tb.cir`: complete testbench: source, DUT instance, analysis, output.
- `sim/tb.par`: testbench parameters generated by Python.
- `sim/sim.py`: generate params, run ngspice, read output, plot.
- `sim/output/`: generated data and plots, ignored by Git except `.gitkeep`.

Read `AGENT.md` and `docs_graph/spice_workflow/` before asking an agent to modify this template.
